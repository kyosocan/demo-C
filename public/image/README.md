## 图片来源

所有图片都需要从 Pixso 设计图中提取：

### 封面图片
- 从资料页面设计图中提取资料卡片的封面图片
- 从学习集页面设计图中提取学习集卡片的封面图片
- 保存为 `封面1.jpg` 到 `封面5.jpg`

### 用户头像
- 从设计图中提取以下用户的头像：
  - 宝贝JOJO
  - 宝贝本周答
  - 英语天天练
  - 金牌数学老师
  - 语文老师
  - 化学老师
  - 英语老师
  - 火火在水星
  - 梦开始的地方
  - 学习培养
  - 我（当前用户）
- 保存到 `avatar/` 目录，文件名与用户名一致

## 使用方法

1. 从 Pixso 设计图中导出图片资源
2. 将封面图片保存到 `public/image/` 目录
3. 将用户头像保存到 `public/image/avatar/` 目录
4. 确保图片文件名与代码中的路径一致

## 注意事项

- 所有图片路径使用 `/image/` 前缀（相对于 public 目录）
- 图片格式建议使用 JPG 或 PNG
- 头像图片建议尺寸：64x64 或 128x128 像素
- 封面图片建议尺寸：400x500 像素或按设计图比例
